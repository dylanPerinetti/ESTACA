## Pour compiler les fichier d'extention .c avec le compilateur GCC.  
Entrez la commande suivante :

```
gcc *c -o main
``` 
### GNU (Linux / MacOs)  
Puis pour executer l'executable depuis votre terminal 
````
./main
````    
### PowerShell (WINDOWS)
Puis pour l'executer depuis votre terminal 
````
main
````
